## Ario Real Estate

Modern real estate landing page built with Next.js and React. Includes modular components for hero, properties, services, neighborhoods, market insights, testimonials, tour gallery, and a final call-to-action.

### Tech Stack
- Next.js (React framework)
- CSS (global styles in `styles/globals.css`)

### Local Development
```bash
npm install
npm run dev
```
Starts at `http://localhost:3000` by default.

### Project Structure (excerpt)
```
components/  # Reusable UI sections
pages/       # Next.js pages (`index.js`, custom `_app.js`, `_document.js`)
styles/      # Global styles
```
